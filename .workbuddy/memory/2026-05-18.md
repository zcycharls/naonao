## 技术要点

### ESM 与 CommonJS 互操作
- CommonJS 中使用 `require()` 无法加载 ESM 模块
- 解决方案: 使用动态 `import()` (返回 Promise)
- 语法: `const module = await import('module-name')`

### Electron 主进程日志转发
- 主进程使用 `webContents.send('channel', data)` 发送消息
- 渲染进程使用 `ipcRenderer.on('channel', callback)` 接收消息
- 通过 `contextBridge.exposeInMainWorld()` 暴露给前端

### transformers.js 本地模型加载
- 设置 `env.localModelPath` 指定本地模型根目录
- 使用模型名称 (如 `'Xenova/Qwen1.5-0.5B-Chat'`) 而不是路径
- 设置 `env.allowRemoteModels = false` 确保完全离线
