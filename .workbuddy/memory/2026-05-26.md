## Body Double 帽子显示/隐藏调试（下午）

### 问题现象
- 点击"📋 一起专注"按钮，帽子不出现，或关闭时不消失
- 根因：`updateBD()` 函数中帽子显示逻辑多次修改未稳定

### 调试过程（共 7 次提交）
1. `69eede5`：用 base64 内联图片绕过 ASAR 路径问题（原始方案）
2. `b68a631`~`f233d5b`：多次尝试修复 `updateBD()` 中 `bd-hat` 的显示/隐藏
3. `f8fdafc`~`bed52d8`：尝试用 opacity/emoji 替代图片，未解决
4. `ed06e15`：改回 `classList.toggle('show')` 方案
5. `1d3b7d2`：改用 `classList.add/remove` + 关闭时清除 `src`
6. `3faf47e`：改用 `style.display` 直接控制
7. `304aa8d`：增加 `setProperty('display','none','important')` 强制隐藏
8. `8c26bd8`：改回 `classList.toggle` + CSS `!important`
9. `f0f3cde`（最新）：完全用 `style.display = bdOn ? "block" : "none"` 控制，移除 class 依赖

### 最终方案（f0f3cde）
- CSS：`.bd-hat { display:none; ... }`（无 `!important`）
- JS：`_h.style.display = bdOn ? "block" : "none"`；`bdOn=true` 时赋值 `HAT_B64`，关闭时清空 `src`
- `HAT_B64` 常量：用 `assets/hat.png` 重新生成正确 base64（之前被 blob 引用污染）
