## 三、发现的 Bug 清单

### 🔴 P0（严重）- 必须修复才能发布

#### Bug #1: Body Double 帽子图片 src 为空
- **位置**: index.html L645, app/index.html L1425
- **严重度**: P0
- **复现步骤**: 
  1. 点击 Body Double 按钮启用功能
  2. 检查帽子图片是否显示
- **预期行为**: 帽子图片应该显示
- **实际行为**: `src=""` 为空，帽子无法显示
- **建议修复**: 
  ```html
  <!-- 需要设置正确的图片路径 -->
  <img class="bd-hat" id="bd-hat" src="assets/hat.png" alt="工作帽">
  ```

#### Bug #2: 番茄钟计时器不精确
- **位置**: index.html L1328-1334
- **严重度**: P0
- **复现步骤**: 
  1. 开始番茄钟
  2. 系统时间被修改或电脑进入休眠
- **预期行为**: 计时器应保持准确
- **实际行为**: 使用 `setInterval(()=>{pomoLeft--}, 1000)` 依赖系统时钟，时间修改会导致计时错误
- **建议修复**: 使用 `Date.now()` 计算时间差，而非简单递减

#### Bug #3: 任务锚（Task Anchor）未持久化
- **位置**: index.html L1260-1270
- **严重度**: P0
- **复现步骤**: 
  1. 设置当前任务
  2. 重启应用
- **预期行为**: 任务应该保留
- **实际行为**: `currentTask` 仅存储在内存变量，重启后丢失
- **建议修复**: 将任务存储到 localStorage

---

### 🟠 P1（重要）- 建议修复

#### Bug #4: Pomodoro 间隔未清理（内存泄漏风险）
- **位置**: index.html L1321-1342
- **严重度**: P1
- **问题描述**: 虽然 `clearInterval` 在 reset/complete 时调用，但如果窗口意外关闭，间隔可能未被清理
- **建议**: 在窗口 unload 事件中添加清理逻辑

#### Bug #5: Freezer 特殊字符未转义
- **位置**: index.html L1483
- **严重度**: P1
- **问题描述**: `freezeIdea()` 直接使用 `item.text`，如果包含特殊字符（如 `"`, `'`）可能导致 JSON 序列化问题
- **建议**: 添加输入验证和转义

#### Bug #6: StatsStore 日期计算可能有时区问题
- **位置**: index.html L1362 (streakDays 函数)
- **严重度**: P1
- **问题描述**: 使用 `toISOString().slice(0,10)` 获取日期，但 `toISOString()` 返回 UTC 时间，可能与本地日期不符
- **建议**: 使用本地日期计算：
  ```javascript
  const today = new Date();
  const localDate = new Date(today.getTime() - today.getTimezoneOffset() * 60000).toISOString().slice(0,10);
  ```

### 🟡 P2（一般）- 可后续修复

#### Bug #7: localStorage 容量未检查
- **位置**: index.html (多个位置)
- **严重度**: P2
- **问题描述**: 如果 localStorage 已满，写入会失败且没有错误处理
- **建议**: 添加 `try-catch` 和容量检查

#### Bug #8: 快速点击 Body Double 按钮可能导致状态不一致
- **位置**: index.html L1467-1470
- **严重度**: P2
- **问题描述**: 没有防抖处理，快速点击可能导致 UI 状态和 localStorage 不一致
- **建议**: 添加防抖（debounce）或禁用按钮 during state change
